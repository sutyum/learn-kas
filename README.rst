Build
=====

.. code-block:: bash

        kas-container build esnp-v1.yaml


Development Shell
=================

.. code-block:: bash

        kas-container shell esnp-v1.yaml

Emulate using Qemu
------------------
This must be carried out from within the kas-container shell
.. code-block:: bash

        runqemu qemux86-64 slirp nographic
